# 🤖 Extensões para Inkscape com IA (GNU GPL v3.0)

## 📌 Sobre o projeto

Este repositório tem como objetivo centralizar a criação, compartilhamento e evolução de extensões para o Inkscape desenvolvidas com auxílio de Inteligência Artificial. A iniciativa surgiu dentro da comunidade **Inkscape BR**, com a proposta de facilitar o acesso a ferramentas úteis e incentivar a colaboração aberta.

## 💡 Propósito

* Reunir extensões em um único lugar
* Incentivar o uso de IA no desenvolvimento de ferramentas gráficas
* Promover colaboração entre designers e desenvolvedores
* Fortalecer o ecossistema de software livre

## ⚙️ Como funciona

As extensões aqui presentes podem ser criadas ou aprimoradas com auxílio de IA, mas sempre revisadas e testadas pela comunidade. O foco é gerar soluções práticas para o dia a dia no Inkscape, mantendo compatibilidade e desempenho.

## 📥 Instalação

1. Baixe a extensão desejada (.zip ou pasta)
2. Extraia os arquivos
3. Copie para a pasta de extensões do Inkscape:

   * Linux: `~/.config/inkscape/extensions/`
   * Windows: `C:\Users\SeuUsuario\AppData\Roaming\inkscape\extensions`
4. Reinicie o Inkscape

## 🚀 Uso

Após instalar, as extensões estarão disponíveis no menu:
**Extensões → [Categoria da Extensão]**

Cada extensão pode conter instruções específicas de uso dentro de sua própria pasta ou documentação.
